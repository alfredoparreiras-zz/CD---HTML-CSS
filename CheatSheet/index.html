HTML & CSS CheatSheet

TAG Description
<HTML> Creates an HTML document.
<!DOCTYPE html> Determine to browser what kind of document it is.
<head> Sets off the title & other info that isn't displayed
<body> Sets off the visible portion of the document
<titler> puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked.
<ul> Creates an unordered list.
<p> Creates a paragraph.
<img src="url"> Adds image; it is a separate file located at the URL
<h1> Creates headlines -- H1=largest, H6=smallest
<br> Inserts a line break (carriage return)
<form> Defines a form
<option> Sets off each menu item
<table> Creates a table
<tr> Sets off each row in a table
<td> Sets off each cell in a row
<h1> Sets off the table header (a normal cell with bold, centered text)

Source: Stanford

Property Description
background Will sets all background styles properties
align-content Will sets the distribution of space between and around content items along a flexbox's cross-axis or a grid's block axis
border Will sets an element's border
box-sizing Will sets how the total width and height of an element is calculated
color Will sets the foreground color value of an element text
cursor Will sets the mouse cursor, if any, to show when the mouse pointer is over an element
flex Will sets how a flex item will grow or shrink to fit the space avaiable in its flex container
font-family Will specifies a prioritized list of one and others (fallbacks) font family names
grid Is a shorthand property that sets all of the explicity and implicit grid properties in a single declaration
height Will sets the height of an element
display The display CSS property sets whether an element is treated as a block or inline element and the layout used for its children, such as flow layout, grid or flex.
overflow The overflow CSS shorthand property sets the desired behavior for an element's overflow — i.e. when an element's content is too big to fit in its block formatting context — in both directions.
padding The padding CSS shorthand property sets the padding area on all four sides of an element at once.
scroll-behavior The scroll-behavior CSS property sets the behavior for a scrolling box when scrolling is triggered by the navigation or CSSOM scrolling APIs.
text-decoration The text-decoration shorthand CSS property sets the appearance of decorative lines on text. It is a shorthand for text-decoration-line, text-decoration-color, text-decoration-style, and the newer text-decoration-thickness property.
transition The transition CSS property is a shorthand property for transition-property, transition-duration, transition-timing-function, and transition-delay.
